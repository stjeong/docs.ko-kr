# <a name="expression-expected"></a>식이 필요합니다.
`+` 또는 `=` 등의 연산자에는 오른쪽 피연산자에 대한 식이 필요하지만 인식할 수 있는 식이 없습니다.  
  
 **오류 ID:** BC30201  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   연산자의 오른쪽 피연산자에 대한 유효한 식을 제공합니다.
