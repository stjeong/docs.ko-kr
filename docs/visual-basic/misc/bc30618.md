# <a name="namespace-statements-can-occur-only-at-file-or-namespace-level"></a>'Namespace' 문은 파일이나 네임스페이스 수준에서만 사용할 수 있습니다.
`Namespace` 문은 `Option` 문, `Imports` 문 및 전역 특성 뒤에, 원본 파일의 다른 모든 선언 앞에 와야 합니다.  
  
 **오류 ID:** BC30618  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   `Namespace` 문을 네임스페이스 선언 또는 원본 파일의 맨 위로 이동합니다.
