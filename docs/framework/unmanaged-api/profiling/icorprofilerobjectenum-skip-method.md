# <a name="icorprofilerobjectenumskip-method"></a>ICorProfilerObjectEnum::Skip 메서드
지정 된 개수의 요소를 건너뛰도록 현재 위치에서이 열거자의 커서를 이동 합니다.  
  
## <a name="syntax"></a>구문  
  
```  
HRESULT Skip (  
    [in] ULONG   celt  
);  
```  
  
#### <a name="parameters"></a>매개 변수  
 `celt`  
 [in] 건너뛸 요소의 수입니다.  
  
## <a name="remarks"></a>설명  
 이 열거자의이 커서의 새 위치는: (현재 위치) + `celt` 합니다.
