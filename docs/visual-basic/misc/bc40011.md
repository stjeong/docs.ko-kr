# <a name="microsoftvisualbasiccomclassattribute-is-specified-for-class-ltclassnamegt-but-it-has-no-public-members-that-can-be-exposed-to-com-therefore-no-com-interfaces-are-generated"></a>클래스의 'Microsoft.VisualBasic.ComClassAttribute'가 지정&lt;classname&gt;' COM에 노출 될 수 있는 public 멤버가 있기 있지만 없으므로 COM 인터페이스가 생성 됩니다
`COMClassAttribute` 특성 블록을 사용하는 클래스는 `Public` 속성 또는 메서드를 정의하지 않습니다. 클래스를 COM 개체로 노출해야 하는 경우 해당 속성과 메서드를 `Public` 액세스 권한으로 선언해야 합니다.  
  
 이 메시지는 기본적으로 경고입니다. 경고를 숨기거나 오류로 처리하는 방법에 대한 자세한 내용은 [Configuring Warnings in Visual Basic](/visualstudio/ide/configuring-warnings-in-visual-basic)을 참조하세요.  
  
 **오류 ID:** BC40011  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   클래스의 하나 이상 속성 또는 메서드에 `Public` 키워드를 추가하거나 `COMClassAttribute` 특성 블록을 제거합니다.
