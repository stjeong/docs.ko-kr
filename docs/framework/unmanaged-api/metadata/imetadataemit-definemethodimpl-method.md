# <a name="imetadataemitdefinemethodimpl-method"></a>IMetaDataEmit::DefineMethodImpl 메서드
인터페이스에서 상속 된 메서드 구현에 대 한 정의 만들고 해당 메서드 구현을 정의에 토큰을 반환 합니다.  
  
## <a name="syntax"></a>구문  
  
```  
HRESULT DefineMethodImpl (   
    [in]  mdTypeDef         td,   
    [in]  mdToken           tkBody,   
    [in]  mdToken           tkDecl  
);  
```  
  
#### <a name="parameters"></a>매개 변수  
 `td`  
 [in] `mdTypedef` 토큰 구현 하는 클래스입니다.  
  
 `tkBody`  
 [in] 합니다 `mdMethodDef` 또는 `mdMethodRef` 코드 본문의 토큰입니다.  
  
 `tkDecl`  
 [in] 합니다 `mdMethodDef` 또는 `mdMethodRef` 구현 되는 인터페이스 메서드의 토큰입니다.
