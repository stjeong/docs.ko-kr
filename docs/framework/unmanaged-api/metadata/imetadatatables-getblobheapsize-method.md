# <a name="imetadatatablesgetblobheapsize-method"></a>IMetaDataTables::GetBlobHeapSize 메서드
(BLOB) 이진 대형 개체 힙 바이트의 크기를 가져옵니다.  
  
## <a name="syntax"></a>구문  
  
```  
HRESULT GetBlobHeapSize (  
    [out] ULONG     *pcbBlobs  
);   
```  
  
#### <a name="parameters"></a>매개 변수  
 `pcbBlobs`  
 [out] 크기 (바이트)는 BLOB 힙에서의 포인터입니다.
