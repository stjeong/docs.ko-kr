<span data-ttu-id="143a6-103">표준 숫자 서식 문자열은 일반 숫자 형식의 서식을 지정하는 데 사용됩니다.</span><span class="sxs-lookup"><span data-stu-id="143a6-103">Standard numeric format strings are used to format common numeric types.</span></span> <span data-ttu-id="143a6-104">표준 숫자 서식 문자열은 `Axx` 형식을 취합니다. 여기서</span><span class="sxs-lookup"><span data-stu-id="143a6-104">A standard numeric format string takes the form `Axx`, where:</span></span>

    > [!NOTE]
    >  <span data-ttu-id="143a6-117">전체 자릿수 지정자는 결과 문자열의 자릿수를 결정합니다.</span><span class="sxs-lookup"><span data-stu-id="143a6-117">The precision specifier determines the number of digits in the result string.</span></span> <span data-ttu-id="143a6-118">선행 또는 후행 공백을 포함한 결과 문자열을 채우려면 서식 항목에서 [복합 서식 지정](../../../docs/standard/base-types/composite-formatting.md) 기능을 사용하고 *맞춤 구성 요소*를 정의합니다.</span><span class="sxs-lookup"><span data-stu-id="143a6-118">To pad a result string with leading or trailing spaces, use the [composite formatting](../../../docs/standard/base-types/composite-formatting.md) feature and define an *alignment component* in the format item.</span></span>
