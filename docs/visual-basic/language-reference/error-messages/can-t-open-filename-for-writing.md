# <a name="cant-open-filename-for-writing"></a>열 수 없습니다. '\<파일 이름 >'을 쓰기용
지정된 된 파일을 쓰기용 이미 열려 있으므로 열 수 없습니다.  
  
 **오류 ID:** BC2012  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
1.  파일을 닫고 다시 엽니다.  
  
2.  파일의 권한을 확인 합니다.
