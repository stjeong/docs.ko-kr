# <a name="target-folder-is-a-file"></a>대상 폴더가 파일입니다.
폴더 관련 작업을 수행하려는 동안 파일을 지정합니다.  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   제공하는 경로를 확인하여 파일이 아니라 폴더를 지정해야 합니다.
