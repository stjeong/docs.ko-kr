# <a name="a-log-has-already-been-created-with-this-name-on-this-machine"></a>이 컴퓨터에 이 이름의 로그가 이미 만들어져 있습니다.
이미 사용 중인 이름으로 로그를 만들려고 했습니다.  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   충돌하는 로그를 제거합니다.  
  
-   로그에 다른 이름을 사용합니다.
