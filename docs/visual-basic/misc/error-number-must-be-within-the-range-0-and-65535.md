# <a name="error-number-must-be-within-the-range-0-and-65535"></a>오류 번호는 0에서 65535 사이의 범위 내에 있어야 합니다.
오류 번호가 0에서 65535 사이인 허용 범위 외부에 있습니다.  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   번호가 유효한 범위 내에 있는지 확인합니다.
