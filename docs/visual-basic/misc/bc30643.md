# <a name="property-propertyname-is-of-an-unsupported-type"></a>속성 '\<propertyname >'는 지원 되지 않는 형식입니다.
참조된 어셈블리의 속성에 표현할 수 없는 형식이 있습니다.  
  
 **오류 ID:** BC30643  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   애플리케이션과 함께 사용할 수 있는 형식을 지정합니다.
