# <a name="internal-error-in-the-microsoft-visual-basic-runtime"></a>Microsoft Visual Basic 런타임에 내부 오류 발생

## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   오류가 발생한 상황을 파악하여 Microsoft 기술 지원 서비스에 문의합니다.
