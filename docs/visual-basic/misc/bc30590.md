# <a name="event-lteventnamegt-cannot-be-found"></a>이벤트 '&lt;eventname&gt;' 찾을 수 없습니다
컴파일러에서 찾을 수 없는 이벤트를 처리하려고 했습니다.  
  
 **오류 ID:** BC30590  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   이벤트 이름의 철자가 올바른지 확인합니다.
