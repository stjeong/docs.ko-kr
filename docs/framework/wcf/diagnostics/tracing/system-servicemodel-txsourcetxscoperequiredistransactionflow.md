# <a name="systemservicemodeltxsourcetxscoperequiredistransactionflow"></a>System.ServiceModel.TxSourceTxScopeRequiredIsTransactionFlow

## <a name="description"></a>설명  
 사용자 코드에 대해 사용 가능한 트랜잭션이 외부 소스에서 이동한 경우 추적됩니다.
