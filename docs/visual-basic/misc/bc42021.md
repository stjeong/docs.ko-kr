# <a name="function-without-an-as-clause-return-type-of-object-assumed"></a>함수에 'As' 절이 없습니다. Object 반환 형식으로 간주합니다.
`Function` 프로시저에서 `As` 절을 지정하지 않습니다.  
  
 `As` 절은 프로그래밍 요소와 연결될 데이터 형식을 식별합니다. 에 [Function 문](../../visual-basic/language-reference/statements/function-statement.md), 값의 데이터 형식을 지정 합니다 `Function` 프로시저가 호출 코드에 반환 합니다. `As` 문에 `Function` 절이 없는 경우 반환 데이터 형식의 기본값은 `Object`입니다.  
  
 이 메시지는 기본적으로 경고입니다. 경고를 숨기거나 오류로 처리하는 방법에 대한 자세한 내용은 [Configuring Warnings in Visual Basic](/visualstudio/ide/configuring-warnings-in-visual-basic)을 참조하세요.  
  
 **오류 ID:** BC42021  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   `As` 문에 `Function` 절을 포함하여 반환 데이터 형식을 지정합니다.
