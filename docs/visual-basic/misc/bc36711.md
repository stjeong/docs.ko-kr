# <a name="argument-matching-parameter-parametername-narrows-to-typename"></a>인수 일치 매개 변수 '\<parametername >'으로 축소 됩니다. '\<typename >'
오버로드된 메서드를 호출했지만 컴파일러에서 축소 변환 없이 호출할 수 있는 메서드를 찾을 수 없습니다. *축소 변환* 은 가능한 값의 일부를 저장하지 못할 수도 있는 데이터 형식으로 값을 변경합니다.  
  
 **오류 ID:** BC36711
