## <a name="example"></a><span data-ttu-id="a44e4-108">예제</span><span class="sxs-lookup"><span data-stu-id="a44e4-108">Example</span></span>  
 <span data-ttu-id="a44e4-109">다음 코드에서는 CS1928 및 CS1929을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="a44e4-109">The following code generates CS1928 and CS1929:</span></span>  
  
```csharp  
// cs1929.cs  
using System.Linq;  
    using System.Collections;  
  
    static class Ext  
    {  
        public static void ExtMethod(this A a)  
        {  
        }  
    }

    {  
    }

    {  
        static void Main()  
        {

    }
```  
