## <a name="communication-types"></a><span data-ttu-id="21842-122">통신 유형</span><span class="sxs-lookup"><span data-stu-id="21842-122">Communication types</span></span>

<span data-ttu-id="21842-123">클라이언트와 서비스는 각각 다른 시나리오 및 목표를 대상으로 지정하여 다양한 유형의 통신을 통해 통신합니다.</span><span class="sxs-lookup"><span data-stu-id="21842-123">Client and services can communicate through many different types of communication, each one targeting a different scenario and goals.</span></span> <span data-ttu-id="21842-124">처음에, 그러한 통신 유형은 두 가지 축으로 분류할 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="21842-124">Initially, those types of communications can be classified in two axes.</span></span>
