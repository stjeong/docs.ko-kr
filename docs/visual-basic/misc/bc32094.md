# <a name="comma-or--expected-type-argument-list"></a>쉼표 또는 ')'가 필요합니다(형식 인수 목록).
형식 인수 목록에 예기치 않은 요소가 포함되어 있습니다. 쉼표(`,`)를 사용하여 이러한 목록에서 형식 인수를 구분하고 닫는 괄호(`)`)를 사용하여 목록을 마칩니다.  
  
 **오류 ID:** BC32094  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   쉼표를 추가하여 형식 인수 목록을 계속하거나 닫는 괄호 문자를 추가하여 목록을 종료합니다.
