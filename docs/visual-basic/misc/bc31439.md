# <a name="partial-method-ltmethodnamegt-cannot-use-the-implements-keyword"></a>부분 메서드 '&lt;methodname&gt;'에서 'Implements' 키워드를 사용할 수 없습니다
부분 메서드(Partial Method)에서 인터페이스 또는 인터페이스 멤버를 구현할 수 없습니다.  
  
 **오류 ID:** BC31439  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
1.  부분 메서드(Partial Method) 정의에서 `Implements` 문을 제거합니다.
