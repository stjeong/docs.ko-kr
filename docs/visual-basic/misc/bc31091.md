# <a name="import-of-type-lttypenamegt-from-assembly-or-module-ltnamegt-failed"></a>형식의 가져오기 '&lt;typename&gt;'어셈블리 또는 모듈의 ' from&lt;이름&gt;' 실패
참조된 모든 라이브러리가 서로 호환되는지 확인합니다.  
  
 **오류 ID:** BC31091  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   참조된 형식을 포함하는 파일이 손상되지 않았는지 확인합니다.
