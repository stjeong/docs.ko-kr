# <a name="icorprofilercallbackmoduleloadstarted-method"></a>ICorProfilerCallback::ModuleLoadStarted 메서드
모듈 로드 되는 프로파일러에 알립니다.  
  
## <a name="syntax"></a>구문  
  
```  
HRESULT ModuleLoadStarted(  
    [in] ModuleID moduleId);  
```

 `moduleId`  
 [in] 로드 되는 모듈의 ID입니다.
