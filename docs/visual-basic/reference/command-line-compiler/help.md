# <a name="-help---visual-basic"></a>-help, -? (Visual Basic)
컴파일러 옵션을 표시합니다.  
  
## <a name="syntax"></a>구문  
  
```  
-help  
' -or-  
-?  
```  
  
## <a name="remarks"></a>설명  
 컴파일에서이 옵션을 포함 하는 경우 출력 파일이 생성 되 고 없는 컴파일을 수행 합니다.  
  
> [!NOTE]
>  `-help` 옵션은 Visual Studio 개발 환경 내에서 사용할 수 있는 명령줄에서 컴파일할 경우에 사용할 수 있는 것입니다.  
  
## <a name="example"></a>예제  
 다음 코드는 명령줄에서 도움말을 표시 합니다.  
  
```  
vbc -help  
```
