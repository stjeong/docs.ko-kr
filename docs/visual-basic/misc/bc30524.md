# <a name="property-ltpropertynamegt-is-writeonly"></a>속성 '&lt;propertyname&gt;'는 'WriteOnly'
`WriteOnly`로 선언된 속성에서 값을 가져오려고 했습니다.  
  
 **오류 ID:** BC30524  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   속성 선언에서 `WriteOnly` 지정자를 제거합니다.
