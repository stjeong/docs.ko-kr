# <a name="loop-control-variable-of-type-lttypenamegt-does-not-implement-the-systemicomparable-interface"></a>형식의 루프 제어 변수 '&lt;typename&gt;'는 'System.IComparable' 인터페이스를 구현 하지 않습니다
변수는 `System.IComparable` 인터페이스를 구현하지 않는 형식입니다. `System.IComparable` 은 값 형식이나 클래스에서 구현되어야 합니다.  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
1.  변수가 값 형식인지 확인합니다.
