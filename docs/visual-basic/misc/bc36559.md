# <a name="anonymous-type-member-property-propertyname-cannot-be-used-to-infer-the-type-of-another-member-property-because-the-type-of-propertyname-is-not-yet-established"></a>무명 형식 멤버 속성 '\<propertyname >' 때문에 다른 멤버 속성의 형식을 유추할 수 없습니다 유형의 '\<propertyname >' 형식이 아직 설정 되지
무명 형식 속성의 형식이 설정될 때까지 다른 속성의 형식을 설정하는 데 사용할 수 없습니다. 예를 들어 다음 선언에서 `.IDName = .LastName` 이 아직 초기화되지 않았으므로 `.LastName` 은 유효하지 않습니다.  
  
```  
' Not valid.   
' Dim anon1 = New With {Key .IDName = .LastName, Key .LastName = "Jones"}   
```  
  
 **오류 ID:** BC36559  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   속성을 사용하기 전에 그 형식을 설정하여 다른 속성을 초기화합니다.  
  
    ```  
    Dim anon2 = New With {Key .LastName = "Jones", Key .IDName = .LastName}  
    ```
