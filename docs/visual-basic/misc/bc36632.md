# <a name="nullable-parameters-must-specify-a-type"></a>Nullable 매개 변수에 형식을 지정해야 합니다.
매개 변수가 nullable 형식의 한정자(?)를 포함하지만 형식을 지정하지 않았습니다.  
  
 **오류 ID:** BC36632  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   nullable 매개 변수를 선언할 때 형식을 지정합니다.
