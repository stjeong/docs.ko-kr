# <a name="index-ltindexnumbergt-for-dimension-ltdimensionnumbergt-is-out-of-range"></a>인덱스 '&lt;indexnumber&gt;'' 차원에 대 한&lt;벗어났습니다&gt;' 범위를 벗어났습니다
인덱스 중 하나가 정의된 범위를 초과했습니다.  
  
 **오류 ID:** BC30702  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   배열 범위를 초과하지 않도록 프로시저의 논리를 변경합니다.  
  
-   광범위한 값을 허용하도록 배열을 다시 정의합니다.
