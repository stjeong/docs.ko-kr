# <a name="itypenamebuilder-interface"></a>ITypeNameBuilder 인터페이스
형식 이름을 작성 하기 위한 메서드를 제공 합니다.  
  
 이 인터페이스는 .NET Framework 인프라를 지원하며 사용자 코드에서 직접 사용할 수 없습니다.
