# <a name="mycomputerports-object"></a>My.Computer.Ports 개체
컴퓨터의 직렬 포트에 액세스하기 위한 속성 및 메서드를 제공합니다.  
  
## <a name="remarks"></a>설명  
 `My.Computer.Ports` 개체의 메서드 및 속성에 대한 자세한 내용은 <xref:Microsoft.VisualBasic.Devices.Ports>를 참조하세요.  
  
 자세한 내용은 [컴퓨터의 포트에 액세스](../../../visual-basic/developing-apps/programming/computer-resources/accessing-the-computer-s-ports.md)합니다.  
  
> [!NOTE]
>  메서드와 속성을 사용할 수도 있습니다는 <xref:System.IO.Ports.SerialPort?displayProperty=nameWithType> 컴퓨터의 직렬 포트에 액세스 하는 클래스입니다.
