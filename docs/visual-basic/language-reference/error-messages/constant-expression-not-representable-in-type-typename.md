# <a name="constant-expression-not-representable-in-type-39lttypenamegt39"></a>상수 식 형식으로 표현할 수 없습니다 &#39; &lt;typename&gt;&#39;
대상 형식에 맞지 않는 상수를 평가 하려는 범위를 오버플로 하는 것 때문에 일반적으로 합니다.  
  
 **오류 ID:** BC30439  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
1.  상수를 처리할 수 있는 대상 유형을 변경 합니다.
