# <a name="connectiongroupmconnectionlist-field"></a>ConnectionGroup.m\_ConnectionList 필드

`ConnectionGroup.m_ConnectionList` 가 <xref:System.Collections.ArrayList> 동일한 URI 및 공유를 제공 하는 연결 개체의 일부 다른 속성 값이 같은 등 만료 인증 합니다.

## <a name="syntax"></a>구문
  
```csharp  
private ArrayList m_ConnectionList
```

> [!WARNING]
> `ConnectionGroup.m_ConnectionList` 필드가 private 이며 코드에서 직접 사용할 하려고 하지 않습니다.
> 
> Microsoft는 어떤 상황에서 프로덕션 응용 프로그램에서이 필드의 사용을 지원 하지 않습니다.

## <a name="requirements"></a>요구 사항

**네임스페이스:** <xref:System.Net>

**어셈블리:** 시스템 (에: System.dll)

**.NET framework 버전:** 2.0부터 사용할 수 있습니다.
