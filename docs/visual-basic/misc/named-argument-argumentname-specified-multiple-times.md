# <a name="named-argument-ltargumentnamegt-specified-multiple-times"></a>명명 된 인수 '&lt;argumentname&gt;' 여러 번 지정 했습니다.
명명된 인수를 두 번 이상 지정했습니다.  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   사양 중 하나만 남기고 모두 제거합니다.
