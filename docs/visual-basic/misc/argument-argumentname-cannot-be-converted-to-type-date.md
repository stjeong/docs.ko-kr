# <a name="argument-argumentname-cannot-be-converted-to-type-date"></a>인수 '\<argumentname >' 'Date' 형식으로 변환 될 수 없습니다.
변환에서 `Date`형식으로 변수를 변경하려고 했지만 값을 유효한 날짜로 인식하지 못합니다.  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   의미 있는 변환을 찾습니다.
