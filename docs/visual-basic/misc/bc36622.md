 <span data-ttu-id="926bc-108">조인 중인 각 컬렉션의 변수 중 적어도 하나는 `Equals` 연산자의 양쪽에서 참조되어야 합니다.</span><span class="sxs-lookup"><span data-stu-id="926bc-108">At least one variable from each collection being joined must be referenced on each side of the `Equals` operator.</span></span>
