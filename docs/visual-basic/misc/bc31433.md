# <a name="method-methodname-cannot-be-declared-partial-because-only-one-method-methodname-can-be-marked-partial"></a>메서드 '\<methodname >' 선언 될 수 없습니다 'Partial' 메서드 하나만 '\<methodname >' 'Partial'를 표시할 수 있습니다
이름과 서명이 같은 두 개의 부분 메서드 선언을 사용할 수 없습니다.  
  
 **오류 ID:** BC31433  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
1.  부분 메서드 선언 중 하나의 이름이나 서명을 변경합니다.
