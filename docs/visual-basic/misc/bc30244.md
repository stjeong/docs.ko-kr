# <a name="ltmodifiergt-is-not-valid-on-a-declare"></a>'&lt;한정자&gt;' Declare 유효 하지 않습니다
`Declare` 문에 `ReadOnly`같은 잘못된 키워드가 포함되어 있습니다.  
  
 **오류 ID:** BC30244  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   `Declare` 문에서 잘못된 키워드를 제거합니다.
