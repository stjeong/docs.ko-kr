# <a name="imetadatatablesgetnextblob-method"></a>IMetaDataTables::GetNextBlob 메서드
테이블의 다음 binary large object (BLOB)의 인덱스를 가져옵니다.  
  
## <a name="syntax"></a>구문  
  
```  
HRESULT GetNextBlob (  
    [in]  ULONG   ixBlob,  
    [out] ULONG   *pNext  
);  
```  
  
#### <a name="parameters"></a>매개 변수  
 `ixBlob`  
 [in] Blob 열에서 반환 된 인덱스입니다.  
  
 `pNext`  
 [out] 다음 BLOB의 인덱스에 대 한 포인터입니다.
