# <a name="ltmethod1gt-and-ltmethod2gt-cannot-overload-each-other-because-they-differ-only-by-parameters-declared-byref-or-byval"></a>'&lt;method1&gt;'및'&lt;method2&gt;'는 'ByRef' 또는 'ByVal'로 선언 된 매개 변수만 다르므로 서로 오버 로드할 수 없습니다.
메서드를 `ByRef` 또는 `ByVal`로 선언된 매개 변수에 의해 첫 번째만 다른 메서드로 오버로드하려고 했습니다.  
  
 **오류 ID:** BC30345  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   `ByRef` 또는 `ByVal` 매개 변수 이름 이외에 메서드에 다른 차별점이 있는지 확인합니다.
