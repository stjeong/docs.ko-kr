# <a name="reference-required-to-assembly-assemblyname-containing-the-definition-for-event-eventname"></a>참조 어셈블리에 필요한 '\<assemblyname >' 이벤트에 대 한 정의가 포함 된 '\<eventname >'
'<`eventname`>' 이벤트에 대한 정의를 포함하는 '<`assemblyname`>' 어셈블리에 대한 참조가 필요합니다. 참조를 프로젝트에 추가합니다.  
  
 이벤트는 프로젝트에서 직접 참조하지 않는 어셈블리 또는 DLL(동적 연결 라이브러리)에서 정의됩니다. Visual Basic 컴파일러는 이벤트가 둘 이상의 DLL 또는 어셈블리에서 정의 되는 경우 모호성을 방지 하기에 대 한 참조가 필요 합니다.  
  
 **오류 ID:** BC30005  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   참조되지 않은 DLL 또는 어셈블리 이름을 프로젝트 참조에 포함합니다.
